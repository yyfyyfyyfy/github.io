Python爬虫爬补天

发表于 2016-08-01   |  

最近在学习Python，想着用Python写一个爬虫把补天的厂商列表爬出来。

首先要把一些模块导入，会用到os模块用来保存路径，re模块来编译正则，requests模块可以做url的请求,也可以从lxml模块导入etree类来获取数据。
程序代码如下：

#导入模块
import os,re,urllib,requests
from lxml import etree

#保存路径的函数
def save_path(path,fileName,message):
    if not os.path.exists(path):
        os.mkdir(path)
    newFileName = fileName+'.txt'
    #拼接存储文件的路径(获取的是存储数据文件的绝对路径)
    newFile = os.path.join(path,newFileName)
    #将列表message 写入本地
    with open(newFile,'a')as f:
        for s in message:
            f.write('%s\t %s\n' % (s[0],s[1]))

def message_info(page):
    dom = etree.HTML(page)
    #在网页中找到厂商名和网址url保存为一个列表
    title = dom.xpath('//tr/td/a/text()')
    url = dom.xpath('//tr/td[2]/text()')

    new_list = []
    for x in range(len(title)):
        new_tuple = (title[x],url[x])
        new_list.append(new_tuple)
    return new_list
#开始爬虫函数
def Spider(url):
    message = requests.get(url).content.decode('utf-8')#解析url，结果保存为utf-8编码
    List = message_info(message)
    message_path = u'补天厂商列表'
    message_fileName = u'补天厂商列表'
    save_path(message_path,message_fileName,List)#调用保存函数

if __name__ == '__main__':
    for p in range(1,5):#循环补天厂商列表的页数
        start_url='https://butian.360.cn/company/lists/page/'+ str(p)
        Spider(start_url)

爬补天的一个爬虫就写好了！！！
